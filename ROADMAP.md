## 🛠️ 기술 스택 추가 고려사항

### Backend
- Redis (세션 관리, 캐싱)
- Celery (백그라운드 작업)
- PostgreSQL (프로덕션 DB)

### Frontend
- Socket.io (실시간 통신)
- TanStack Query (상태 관리)
- Zustand (전역 상태)

### DevOps
- Docker 컨테이너화
- GitHub Actions (CI/CD)
- Nginx (리버스 프록시)
- SSL 인증서
