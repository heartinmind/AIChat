# Cursor에게 지시할 내용 📋

## 1️⃣ 패키지 설치 (이것만 복사해서 Cursor에 붙여넣으세요)

```
터미널에서 requirements.txt의 모든 패키지를 설치해주세요:
pip install -r requirements.txt
```

## 2️⃣ 서버 실행 (설치 후)

```
web_server.py를 실행해주세요:
python web_server.py
```

## 3️⃣ 문제 발생 시

```
방금 발생한 에러를 해결해주세요
```

---

## 💡 Cursor 터미널 단축키
- **터미널 열기**: `Ctrl + `` (백틱)
- **새 터미널**: `Ctrl + Shift + ``
- **터미널 닫기**: 터미널에서 `exit`

## 🔧 자주 사용하는 Cursor 명령어

### 파이썬 패키지 관련:
- "requirements.txt 설치해줘"
- "fastapi 설치해줘"
- "설치된 패키지 목록 보여줘"

### 서버 실행:
- "웹서버 실행해줘"
- "5000번 포트로 서버 시작해줘"
- "서버 중지해줘" (Ctrl+C)

### 문제 해결:
- "ModuleNotFoundError 해결해줘"
- "포트 사용중 에러 해결해줘"
- "파이썬 버전 확인해줘"
